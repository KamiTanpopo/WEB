Використання атрибута id

HTML атрибут id використовується для вказівки унікального ідентифікатора HTML елемента.
В межах всієї HTML-сторінки повинен існувати тільки один тег з певним значенням атрибута id.
Наявність декількох елементів з однаковим значенням id не допускається і може привести до помилки.

При описі ідентифікатора спочатку вказується символ решітки (#), потім йде ім'я ідентифікатора.
