Опис предметного середовища:

Цей сайт створений для бронювання квитків в мережі кінотеатрів.

Хедер :

- реєстрація або вхід;
- про кінотеатр та компанію;
- найближчі кінотеатри;
- відгуки, рекомендації, скарги що стосуються створеного сайту;
- оплата онлайн через карту;

Основне тіло сайту:

- aфіши прем’єр та їх опис
- дата, вартість сеансу
- бронювання
- Правила поводження в кінотеатрі

Футер:

- контакти (номер телефону, електронна пошта, телеграм...)
- допомога
- соціальні мережі
- Графік роботи .

Функціонал:

- Перелік всіх фільмів, мультфільмів та їх опис та постери ;
- Сортування за жанрами, за популярністю, датою виходу, країною, що зняла фільм, отримали престижні нагороди ;
- Підбір за місцем розташування кінотеатрів
- Оренда залу для закритого користування
- Список сеансів фільмів найближчі за часом
- Реєстрація на сайті
- Бронювання квитків онлайн:
  • Вибір залу
  • Вибір місця
  • Вибір фільму
- Попереднє замовлення їжі та напоїв задля економії часу перед фільмом
- Онлайн допомога користувачу
- Вибір мови
- Форум
- Реклама
